## Node API with JWT auth & Pagination

### Features
Full fledged node (express) restful api with JWT Authentication & Pagination.

### Up and running:

First install the dependencies using, npm install

1. node server.js or nodemon server.js (npm install -g nodemon)
2. /setup (to create new user)
3. /authenticate (to generate token)
4. /api/jokes?token=token_value (for every request)

Change the db in the config.js
